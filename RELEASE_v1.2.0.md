## ✅ 更新内容

### 🚀 字典词组前缀匹配 (v1.2.0 新增)

#### 改进点
阶段4的字典词组匹配从**精确匹配**改为**前缀匹配**，允许部分拼音输入匹配词组。

**示例**:
```
输入"zhg":
v1.1.0: 无法匹配"中国" ❌
v1.2.0: "中国"在第1页 ✅

输入"bjs":
v1.1.0: 无法匹配"北京市" ❌
v1.2.0: "北京市"在第1页 ✅
```

#### 匹配优先级（8级）
| 优先级 | 阶段 | 说明 |
|-------|------|------|
| 1 | 用户词库单字匹配 | 用户词库单字，code 前缀匹配 |
| 2 | 单字精确匹配 | 字典单字，code 完全匹配 |
| 3 | 用户词库词组匹配 | 用户词库词组，code 前缀匹配 |
| 4 | **词组前缀匹配** | **字典词组，code 前缀匹配 (v1.2.0 改进)** |
| 5 | 用户词库简拼匹配 | 用户词库按声母匹配 |
| 6 | 简拼匹配 | 字典词组按声母匹配 |
| 7 | 简拼+后字匹配 | 如 `zhguo`→"中国" |
| 8 | 逐字匹配 | 前缀匹配兜底 |
