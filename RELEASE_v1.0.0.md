# Release v1.0.0

**发布日期**: 2026-07-08

这是 Cardputer IM 拼音输入法固件的首个正式发布版本。

## 主要特性

### 🎯 拼音输入引擎

- **5级匹配优先级**: 用户词库 → 精确匹配 → 简拼匹配 → 简拼+后字匹配 → 逐字匹配
- **精确匹配优先**: 输入`zhong`直接显示"中"，不被简拼覆盖
- **智能简拼**: `zg`匹配"中国"、"总共"，`zh/ch/sh`保持两字母
- **��合匹配**: `zhguo`匹配zh简拼+guo完整拼音→"中国"

### 📊 动态词频

- **单字+词组支持**: 所有词条都支持频次调整
- **自动学习**: 选择后自动增加频次，常用词条自动置顶
- **持久化**: SPIFFS存储，最多1000词条，掉电不丢失

### 🔧 用户词库管理

- **v键删除模式**: 按`v`显示所有用户词条，选择即删除
- **DD标记**: 类似两分输入的LF标记，直观易用
- **完整API**: 支持清空、批量删除等操作

### 📚 海量词库

- **74,636词组** + **6,750单字**
- **GB2312完整覆盖**: 所有词库内容在GB2312范围内
- **2MB Flash存储**: 通过mmap读取，不占用RAM

### ⚡ 无PSRAM优化

- **内存占用小**: 前缀索引仅2.7KB，用户词库最多8KB
- **Flash存储**: 词库存储在Flash，运行时按需读取
- **候选词限制**: 最多100个候选词，避免内存溢出

## 刷写方法

### 完整固件（新设备）

```bash
esptool --chip auto --port /dev/ttyUSB0 --baud 921600 \
  --before default_reset write_flash -z 0x0 typewriter-pinyin-v1.0.0.bin
```

### 纯净固件（升级）

```bash
esptool --chip auto --port /dev/ttyUSB0 --baud 921600 \
  --before default_reset write_flash -z 0x10000 firmware-pinyin-v1.0.0.bin
```

## 使用示例

### 基本输入

```
输入: z        → z开头单字（早、在、做...）
输入: zh       → zh开头单字（中、这、着...）
输入: zhong    → 中（精确匹配优先）
输入: zg       → 中国、总共（简拼匹配）
输入: zhguo    → 中国（zh简拼+guo完整拼音）
```

### 动态词频

```
第1次输入zhong → 选择"中" → 频次=1
第2次输入zhong → "中"置顶 → 选择"中" → 频次=2
第3次输入zhong → "中"依然置顶（频次最高）
```

### 删除模式

```
按v键          → DD: 中 中国 总共...（显示所有用户词条）
输入zhong      → DD:zhong 中（过滤匹配）
选择1          → 删除"中"，不上屏，自动退出
```

## 技术规格

- **硬件**: M5Stack Cardputer ADV (ESP32-S3)
- **Flash需求**: 至少8MB
- **PSRAM**: 不需要
- **RAM占用**: 44.2% (141KB/320KB)
- **Flash占用**: 87.3% (4.48MB/5MB)

## 已知问题

- Flash空间占用较高（87%），如需更多空间可精简词库
- 用户词库最多1000条，超出后FIFO淘汰最旧词条
