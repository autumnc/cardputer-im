## ✅ 包含的修复

此固件包含**所有P0和P1级别的安全修复**:

### P0级别 - 严重问题修复 ✅

#### 1. EPD内存分配失败死循环修复 ✅
- **问题**: PSRAM分配失败后设备永久锁死在`while(1)`循环
- **修复**: 显示错误信息,优雅退出,允许用户重启恢复
- **文件**: `src/display/EPD/display_EPD.cpp`

#### 2. WiFi DMA缓冲区管理改进 ✅
- **问题**: DMA分配失败后可能导致WiFi操作崩溃
- **修复**: 添加`wifi_available`标志,优雅禁用WiFi功能
- **文件**: `src/app/app.cpp`, `src/service/Sync/Sync.cpp`

#### 3. 多核心竞态条件消除 ✅
- **问题**: Editor类buffer访问缺少锁保护,可能导致数据损坏
- **修复**: 为`getBufferSize()`和`resetBuffer()`添加RAII锁
- **文件**: `src/service/Editor/Editor.h`

### P1级别 - 安全和稳定性修复 ✅

#### 4. IME用户词库DoS防护 ✅
- **问题**: 单行无长度限制,恶意文件可耗尽内存
- **修复**: 添加256字节单行长度限制,改进字节统计
- **文件**: `src/service/IME/IME.cpp`

#### 5. 文件保存整数溢出保护 ✅
- **问题**: `seekPos + loadedLength`可能溢出导致数据损坏
- **修复**: 添加溢出检查,防止大文件处理崩溃
- **文件**: `src/service/Editor/Editor.cpp`

#### 6. WiFi配置加载验证 ✅
- **问题**: 无文件大小限制,解析失败直接删除配置
- **修复**: 4KB文件大小限制,损坏文件备份而非删除
- **文件**: `src/service/WifiEntry/WifiEntry.cpp`

#### 7. sync_send内存检查溢出修复 ✅
- **问题**: `fileSize + SYNC_HEAP_MARGIN`可能溢出
- **修复**: 分离检查,避免整数溢出,详细错误信息
- **文件**: `src/service/Sync/Sync.cpp`

#### 8. IME内存碎片化优化 ✅
- **问题**: 频繁`shrink_to_fit()`导致堆碎片化
- **修复**: 提高阈值,预分配合理容量,减少碎片
- **文件**: `src/service/IME/IME.cpp`

## 🔧 刷写说明

### 方法1: 使用M5Burner(推荐)
1. 下载 `typewriter-pinyin-v1.0.1-p1-fixed.bin`
2. 打开M5Burner
3. 选择设备: M5Cardputer
4. 选择固件文件并刷写

### 方法2: 使用esptool命令行
```bash
esptool --chip auto --port /dev/ttyUSB0 --baud 921600 \
  --before default_reset write_flash -z 0x0 \
  typewriter-pinyin-v1.0.1-p1-fixed.bin
```

**注意**: 将 `/dev/ttyUSB0` 替换为您的实际串口设备

## 🛡️ 安全性改进

### 防御的攻击向量
1. ✅ **设备死锁**: PSRAM分配失败不再卡死设备
2. ✅ **DoS攻击**: 恶意用户词库/WiFi配置无法耗尽内存
3. ✅ **内存耗尽**: 配置文件大小限制(4KB)
4. ✅ **整数溢出**: 文件偏移和内存检查防止溢出
5. ✅ **数据损坏**: 多核心竞态条件消除,文件保存检查
6. ✅ **数据丢失**: 配置文件损坏时备份而非删除

### 稳定性提升
- ✅ 大文件处理更安全
- ✅ 内存碎片化减少
- ✅ 长期运行稳定性提升
- ✅ 错误信息更明确详细
- ✅ WiFi功能不可用时优雅降级

## ⚠️ 重要提示

### 与旧版本的差异
此固件包含重要的安全修复,**强烈建议**从旧版本升级:

| 场景 | v1.0.0行为 | v1.0.1行为 |
|------|-----------|-----------|
| PSRAM分配失败 | 永久死锁 | 显示错误,可重启 |
| WiFi DMA失败 | 可能崩溃 | WiFi禁用,仍可编辑 |
| 多核心buffer访问 | 可能数据损坏 | 线程安全,数据一致 |
| 超长字典行 | 内存耗尽 | 跳过并记录 |
| 大文件偏移 | 可能崩溃 | 检查并报错 |
| WiFi配置损坏 | 直接删除 | 备份并提示 |
| 内存碎片 | 累积增长 | 显著减少 |
